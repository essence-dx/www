
# dx-www-browser-micro

Micro browser WASM runtime for dx-www - NO_STD Integer DOM (< 2 KB).

## Overview

This crate provides an extremely small browser WASM runtime with zero dependencies, targeting sub-2KB bundle sizes for performance-critical applications.

## Installation

```toml
[dependencies]
dx-www-browser-micro = { path = "../browser-micro" }
```
