
# dx-www-browser

Sub-20KB browser WASM runtime for dx-www - pure FFI, no wasm-bindgen.

## Overview

This crate provides the main browser-side WASM runtime for dx-www applications, optimized for minimal bundle size.

## Installation

```toml
[dependencies]
dx-www-browser = { path = "../browser" }
```

## Features

- `dev`
- Development mode with error handling and debugging
