# DX Style Tailwind Compatibility Matrix

This matrix records what dx-style can honestly claim today. It is not a promise of full Tailwind parity.

Latest Tailwind truth verified for this matrix: `tailwindcss@4.3.0` via `npm view tailwindcss version` on 2026-05-24. dx-style is not a complete Tailwind v4.3 replacement.
